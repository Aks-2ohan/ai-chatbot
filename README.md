# AI Chatbot

An AI-powered chatbot that answers user queries through interactive 
conversational responses, built using HTML, CSS, and JavaScript.

## Features
- Real-time AI responses using Google Gemini API
- Natural Language Processing for understanding user queries
- Conversation memory — remembers previous messages in the chat
- Error handling for failed API calls
- Clean and responsive user interface
- Send message by clicking Send or pressing Enter

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Gemini API (AI/ML + NLP)
- REST API Integration

## How to Run
1. Clone this repository
2. Get a free API key from https://aistudio.google.com/app/apikey
3. In script.js, replace YOUR_API_KEY_HERE with your actual key
4. Open index.html with Live Server in VS Code

## Project Structure
- index.html — User interface and chat layout
- style.css — Styling and responsive design
- script.js — API integration and chat logic
